## 알고리즘 기본기 다지기📚

코딩테스트 준비에 앞서 기본문법부터 차근차근 풀어보기 위해서 [코드업](https://codeup.kr/problemsetsol.php?psid=23)의 기초 100제를 풀어보는 repository이다.

구현언어는 python 이지만 문제에 따라 C++도 참고할 수도 있다.
